You want to add up the numbers from 0 to 9 and write the following code to do so. What (if anything) is wrong with this code?
int i;
int sum=0;
for (i=0; i<10; i--) {
  sum = sum+i;
}
printf("The sum of the number from 0 to 9 equals %d\n", sum);
  1. The variable i needs to be initialized.
  2. There is nothing wrong with this code.
  3. One needs to use a while-loop here instead of a for-loop.
  4. The instruction i-- should be i++
