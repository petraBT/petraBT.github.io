Solution 6.3.1.

Remember from the video above that binary is base-2, with the least significant bit as the right-most digit, which has a place value of \(2^0\text{.}\) Moving left, the the place values are \(2^1\text{,}\) \(2^2\text{,}\) \(2^3\text{,}\) etc. Only those digits that are “1” contribute their place value to the number, so for this binary number, we interpret it as

\begin{equation*} (0 × 2^7) + (1 × 2^6) + (0 × 2^5) + (0 × 2^4) + (0 × 2^3) + (0 × 2^2) + (0 × 2^1) + (1 × 2^0) = 64 + 1 = 65 \end{equation*}
