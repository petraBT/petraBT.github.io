Solution 5.2.1.

In unsigned binary, 10000001 represents

\begin{equation*} (1 × 2^7) + (0 × 2^6) + (0 × 2^5) + (0 × 2^4) + (0 × 2^3) + (0 × 2^2) + (0 × 2^1) + (1 × 2^0) = 128 + 1 = 129 \end{equation*}

In signed magnitude binary, 10000001 represents

\begin{equation*} -1 * [(0 × 2^6) + (0 × 2^5) + (0 × 2^4) + (0 × 2^3) + (0 × 2^2) + (0 × 2^1) + (1 × 2^0)] = -1 * (1) = -1 \end{equation*}

since the most significant bit (the left-most bit) is "1", which denotes a negative number in this binary representation. Thus, that one binary number represents 129 in binary and -1 in signed magnitude, which are definitely different!
