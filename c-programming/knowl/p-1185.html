  • The address of a variable has a datatype that depends on the datatype of the variable itself. For example, the address of an int has datatype int * (also called integer pointer). Example: int *addr; declares a pointer named "addr"

  • Use & to get the address of a variable. Example: *addr = &myVar stores the address of myVar in *addr

  • To print an address, use %p as the format specifier
