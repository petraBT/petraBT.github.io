Though this task could technically be accomplished with either type of loop, a while-loop would be more efficient to use, since you don't know how many times the user will input information before that certain "stop" integer is entered. With a for-loop, you'd have to prescribe a certain number of iterations through the loop, but you don't know this number up front! A while-loop does not need to be told how many times to iterate, rather it can simply run until it finds that certain "stop" integer being entered.
