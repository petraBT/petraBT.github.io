Paragraph

  • Output using format specifiers can be formatted, using the syntax:
%[flag][width][.precision][type_mod]type
  • Parameters in brackets [] are optional.
  • The [flag] can take on values such as -, +, 0 etc. For example, - (as in %-d) will cause the numerical output to be left-justified.
  • The [.precision] option can be used to specify, for a floating point number, the number of digits after the decimal point to be displayed. For example, %.3f will print three digits after the decimal point.
  • The [width] option can be used to prescribe the number of spaces that the printed number will take up. The [flag] can determine how these numbers appear (i.e. left-justified).
