It is even possible to define a new data type and associate it with our new data structure. The C-command to do so is typedef. The code window below shows the use of typedef in order to assign the name student_t to the more cumbersome student structure struct student. This way we can use the new data type student_t just like any of the C pre-defined data types int, float, double, char, void of which we know so far.
