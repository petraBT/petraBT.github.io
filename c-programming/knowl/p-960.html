Notice how the fscanf() function communicates with us in two ways: Values read from the file are transmitted to us via the variables (with ampersands in front of them) that appear inside the parentheses (in this case we call the variable num). On the other hand, the return value (which we store in the variable result here) signifies whether the end of the file has been reached. When this is the case, the return value equals EOF.
