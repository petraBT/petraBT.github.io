There are multiple ways to customize a script file so as to respond to a user's needs. Since the script file shares the variable workspace with the command window, one way is to simply create a variable in the command window and assign it a certain value before running the script file. That value can then be used within the script file. But that's not a very clean way to go about this. A better method is to have the script ask for user input or read input from a file, for example via the fscanf() command.
