Item 2.

Assign values to the variables in the Command Window and then run the script as demonstrated below. In this example, the script is named doStuff.m. If this script processes data from a file whose name is stored in the variable filename then one way to change from which file the script reads is to assign a filename in the Command Window before running the script.
>> filename = 'numbers.txt'
>> doStuff
>> filename = 'more_numbers.txt'
>> doStuff
